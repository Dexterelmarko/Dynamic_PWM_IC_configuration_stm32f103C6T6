# Dynamic_PWM_IC_configuration_stm32f103C6T6

[![Platform: STM32](https://img.shields.io/badge/platform-STM32-blue)](https://www.st.com)
[![IDE: STM32CubeIDE](https://img.shields.io/badge/IDE-STM32CubeIDE-green)](https://www.st.com)

## О проекте
Данный проект реализует динамическую конфигурацию портов микроконтроллера stm32f103C6T6 для работы в режиме генерации или измерения ШИМ сигналов. 

Передача команд конфигурации портов и параметров работы осуществляется по интерфейсам USB и UART.

## Возможности
- Генерация ШИМ сигналов с заданными частотой и скважностью
- Измерение ШИМ сигнала с заданными параметрами (размер окна измерений, тип фильтра, время ожидания)
- Просмотр действующей распиновки и конфигурации портов

## Аппаратная часть

### Микроконтроллер
- **Модель:** STM32F103C6T6 (Blue Pill)
- **Тактовая частота:** 72 МГц (внешний кварц 8 МГц)

### Используемые периферийные блоки
- USB (основной интерфейс)
- USART1 (основной интерфейс, скорость 115200 бод)
- SPI1 (для подключения модуля W5500)
- TIM1, TIM2, TIM3 (для конфигурации портов с поддержкой таймера)

## Архитектура проекта

### Описание модулей (Файловая структура)

- **`Core/`** – Основная логика прошивки: main.c, обработка прерываний, инициализация тактирования и пинов, обмен данных по интерфейсам.
- **`Drivers/`** – HAL драйверы, CMSIS-ядра, W5500-стек. Отвечают за низкоуровневую работу с регистрами.
- **`Middlewares/ST/STM32_USB_Device_Library/`** – Кроссплатформенный стек USB Device Library. Содержит классы (CDC, HID) и ядро обработки USB-событий.
- **`USB_DEVICE/`** – Конкретная реализация USB-приложения. Здесь генерируются файлы `usbd_desc.c` (дескрипторы) и `usbd_cdc_if.c` (функции приёма/передачи для CDC-класса).
- **`main.c`** – Инициализация тактирования, HAL, функции динамической конфигурации портов, фильтрации измерений, обработки прерываний.
- **`command_handler.c/h`** – Обработка запросов от интерфейсов, запуск функций конфигурации, формирование и отправка ответов.

### Описание основных функций файла main.c

- **`PWM_config`** – Конфигурация порта с поддержкой таймера в режим генерации ШИМ с выбранными значениями частоты и скважности.
- **`PWM_EMU_config`** – Конфигурация порта без поддержки таймера в режим генерации ШИМ с выбранными значениями частоты и скважности.
- **`IC_config`** – Конфигурация порта с поддержкой таймера в режим измерения ШИМ.
- **`IC_config`** – Конфигурация порта с поддержкой таймера в режим измерения ШИМ.
- **`IC_EMU_config`** – Конфигурация порта без поддержки таймера в режим измерения ШИМ.
- **`IC_freq`** – Измерение и фильтрация сигнала.
- **`HAL_TIM_PeriodElapsedCallback`** – Обработка прерывания для генерации ШИМ на порту без поддержки таймера.
- **`HAL_TIM_IC_CaptureCallback`** – Измерения сигнала на порту с поддержкой таймера.
- **`HAL_GPIO_EXTI_Callback`** – Измерения сигнала на порту без поддержки таймера.

### Описание основных функций файла command_handler.c

- **`Parse_package`** – Обработка запросов, активация функций конфигурации и формирование ответа.
- **`Send_Answer`** – Отправка ответа по последнему интерфейсу.

## Описание протокола
Протокол связи реализован по принципу запрос-ответ в виде CLI консоли.

### Описание команд (запросов)

| Команда   | Параметры       |  Описание                     |     Пример           |
|-----------|-----------------|-------------------------------|----------------------|
| **help**  | нет   		  |Выводит список доступных команд|     help             |
| **echo**  | нет   		  |Возвращает переданный текст    |     echo             |
| **pinout**| нет   		  |Показывает состояние пинов     |     pinout           |
| **config_gen**| <пин> <частота Гц> <скважность %>   		  |Настройка пина для генерации ШИМ     |     config_gen PA8 5600 25  |
|  | <пин>   		  |Название порта заглавными буквами    |      PA8  |
|  | <частота Гц>     |Частота ШИМ (от 1 до 10000 Гц)       |      5600 |
|  | <скважность %>   |Скважность ШИМ (от 0 до 100 %)       |      25   |
| **gen** | <состояние>   |Запуск или остановка ШИМ       |      gen start   |
|  | <состояние>    | start - Запуск, stop - Остановка      |         |
| **config_measure** | <пин>   |Настройка пина для измерения частоты      |      config_measure PA7   |
|  | <пин>    | Название порта заглавными буквами     |   PA7    |
| **measure_params** | <окно> <фильтр> <таймаут>  |Настройка параметров измерения      |      measure_params 10 median 500  |
|  | <окно>    | Окно фильтрации (от 1 до 100 отсчетов)     |   10    |
|  | <фильтр>    | Тип фильтрации (none, mean, median, min, max)     |   median    |
|  | <таймаут>    | Максимальное время ожидания измерения     |   500    |
| **freq** | нет   |Возвращает измерение      |      freq   |

### Типы фильтрации измерений

| Фильтр   | Функция в main.c       |  Описание                     |
|-----------|-----------------|-------------------------------|
| **mean**  | **mean_filter()**    |Сглаживает случайный шум. Усреднение N измерений.|
| **median**  | **median_filter()**    |Удаляет импульсные выбросы. Берёт серединное значение.|
| **min**  | **min_filter()**    | Ищет минимальное значение за окно (огибающая снизу).|
| **max**  | **max_filter()**    |Ищет максимальное значение (огибающая сверху).|
